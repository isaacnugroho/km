# km

A knowledge management system.

Learning Ontologies and the workspace Case Ontology share the same **runtime vs export** model: Git-ignored quad-stores (`.km/`, `{source}/lo_quads.db`) and Git-tracked Turtle exports (`{source}/exports/`, workspace `case-exports/`). See [docs/knowledge-management-specification.md](docs/knowledge-management-specification.md) §2.6 for Case exports and §2.4–2.5 for LO packages.
